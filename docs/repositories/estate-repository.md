# Estate Repository

## Estates
```php
use Innobrain\OnOfficeAdapter\Facades\EstateRepository;

$estates = EstateRepository::query()
    ->get();

$estate = EstateRepository::query()
    ->first();

$estate = EstateRepository::query()
    ->find(1);

EstateRepository::query()
    ->each(function (array $estates) {
        // First page
    });

EstateRepository::query()
    ->addModify('estate_id', 1)
    ->modify(1);

$estate = EstateRepository::query()
    ->create([
        'estate_id' => 1,
    ]);
```
